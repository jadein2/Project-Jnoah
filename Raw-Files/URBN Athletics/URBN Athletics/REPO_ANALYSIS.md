## 1. Workspace Summary

This workspace is a brand and operations documentation repo for the rebrand from Urban Strong Fitness to URBN Athletics.

The repo is not a software codebase. It is a business operating and brand knowledge base with:

- Rebrand strategy
- Business overview
- Operations manual
- Audit report
- Meeting minutes
- Announcement graphics
- Facility / equipment schematic images

The central decision across the source files is clear:

> URBN Athletics is the new public-facing brand. Urban Strong Fitness becomes the legacy foundation and origin story.

## 5. Consolidated Brand Decision

Locked:

- Main public brand: URBN Athletics
- Former name: Urban Strong Fitness
- Former name role: origin story / legacy foundation
- Tagline: Built to Move Better
- Rebrand type: name-change rebrand only
- Core position: coached fitness gym for busy adults
- Core differentiator: coaching over access
- Business model: coaching-first, not cheap access-only
- Public reassurance: same coaching, same community, same standard
- Restart Fitness PH: do not touch

Not locked yet:

- Final logo
- Final color palette
- Final typography
- Final social templates
- Final website visual direction
- Final Pioneer agreement
- Final PFT standards
- Final On-Ramp scheduling system

## 6. Main Risk Found

The repo has multiple large documents with overlapping brand language.

The risk is that future assets may pull from the older Urban Strong naming instead of the URBN Athletics rebrand direction.

Fix implemented:

- Added `Main-Brand-Brain.md` as the canonical brand source of truth.
- Added `Brand-Kit.md` as the practical execution guide.
- Added this `REPO_ANALYSIS.md` file to map the workspace and source hierarchy.
