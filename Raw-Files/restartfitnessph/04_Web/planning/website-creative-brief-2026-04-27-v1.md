# Restart Fitness PH Website Creative Brief

Status: Active draft  
Created: 2026-04-27  
Target domain: `www.restartfitnessph.com`  
Build platform: Wix Studio  
Role: Main creative team project alongside social media marketing

## Website Goal

Create a premium, friendly, energetic, and empathetic website for Restart Fitness PH that helps people understand The Restart Fitness System and take the next step toward a Discovery Call.

## Primary CTA

Book a Discovery Call

## Secondary CTA

DM "RESTART"

## Initial Page Priority

1. Homepage
2. Coaching / The Restart Fitness System
3. Results / Proof
4. About Coach Jap
5. Discovery Call / Start Here

## Homepage First-Pass Structure

1. Hero
   - Clear promise
   - Brand identity
   - Book a Discovery Call CTA
   - Trust-building visual

2. Problem
   - People are putting in effort but not seeing real progress
   - Avoid shame-based framing
   - Position structure as the missing piece

3. System
   - The Restart Fitness System
   - Structure, practical guidance, accountability
   - Simple steps or pillars

4. Proof
   - Client results or testimonials only when consent is confirmed
   - Use gold only for proof/results contexts

5. Offer
   - Coaching path
   - Who it is for
   - What happens next

6. CTA
   - Book a Discovery Call
   - Optional social CTA: DM "RESTART"

## UI/UX Priorities

- Mobile-first layout
- Clear navigation
- Fast understanding above the fold
- Premium but not cold
- Friendly without looking casual or generic
- Energetic without shouting
- Strong CTA visibility
- Wix Studio-ready sections and responsive layouts
