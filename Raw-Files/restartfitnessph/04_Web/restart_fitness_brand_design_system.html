Typography system — Golden Ratio Scale
Type is the structure.
Base unit: 16px. Each step multiplied by φ = 1.618. Cormorant Garamond Italic for all display/headline text. DM Sans for all body, UI, and structural copy. Never swap these roles.
Golden ratio type scale (φ = 1.618)
Scale step Size Role Bar
φ⁴ — Display 68px Campaign hero text
φ³ — Headline 42px Page H1, section hero
φ² — Title 26px Section titles, H2
φ¹ — Subhead 20px Sub-sections, intro text
φ⁰ — Body 16px All body copy
φ⁻¹ — Label 10px Overlines, tags, captions

Direct. Professional.
Uncompromising.

Tone guidelines
Y
Direct — "Here's the process. Let's execute."
Y
Confident — state facts, not motivational filler
Y
Empathetic — acknowledge the struggle without shame
Y
Clear — one idea per sentence, one CTA per post
Y
Selective — "This is not for everyone" is a feature
N
Never use "the last first step" — permanently banned phrase
N
Never focus messaging on "weight loss" as the primary goal
N
Never use "restart" as a verb in any sentence
N
Never use shame-based, macho, or intimidation language
N
Never use "tailored" — use "structured" or "built for"
N
Never use fake urgency without substance behind it
