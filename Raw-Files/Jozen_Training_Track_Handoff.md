# VARIANCE RULE

Do not repeat:
- Same strength pattern
- Same conditioning format
- Same conditioning sequence

# HYPERTROPHY BLOCK RULES

Cycle 2 and beyond:
- More accessory volume
- More unilateral work
- More upper back work
- More chest volume
- More shoulder volume

# MOBILITY RULES

# THINGS TO AVOID

# QUALITY CHECK

Before creating any session:

1. Is this different from the previous day?
2. Is the conditioning format different?
3. Is the movement emphasis different?
4. Does accessory work support strength work?
5. Is RPE 7-8 maintained?
6. Does the session improve strength, muscle, or work capacity?
