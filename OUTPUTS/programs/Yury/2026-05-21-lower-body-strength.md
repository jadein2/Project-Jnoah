STRENGTH BLOCK A — SQUAT + PULL | 4 Rounds
Perform A1. Complete A2 immediately after each set of A1. Rest 90 sec before returning to A1.

A1 — Back Squat 5 x 4 sets @8 RPE
*Full squat as much as possible. Up the weight if perceived light.

A2 — 4 Pull-Ups after each squat set

STRENGTH BLOCK B — LOWER ACCESSORY | 3 Rounds
Perform B1 → B2 back to back. Rest 60–90 sec after B2.

B1 — Bulgarian Split Squat 10 x 3 sets (each leg) @2xDB
*Control the descent. Front shin vertical. Full range at the bottom.

B2 — Leg Press 12 x 3 sets @8–9 RPE

STRENGTH BLOCK C — HIP + POSTERIOR CHAIN | 3 Rounds
Perform C1 → C2. Rest 60 sec after C2.

C1 — Hip Adductor Machine 12 x 3 sets @8–9 RPE
