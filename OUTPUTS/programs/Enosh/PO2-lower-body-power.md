Specific Warm-Up

STRENGTH BLOCK A — SQUAT + POWER | 4 Rounds
Perform A1 then A2. Rest 90 sec after A2.

A1 — Back Squat 6 x 4 sets @8 RPE
*Match or add 2.5–5kg from C1. Full depth. Record load.

A2 — Broad Jump 5 x 4 sets @max effort
*Full hip extension. Land soft — absorb with both legs. Reset before each rep.

STRENGTH BLOCK B — HINGE + SINGLE LEG | 3 Rounds
Perform B1 then B2. Rest 90 sec after B2.

B1 — Romanian Deadlift 8 x 3 sets @7–8 RPE
*3-count eccentric. Full hamstring stretch. Bar close to legs.

B2 — Bulgarian Split Squat 8 x 3 sets (each leg) @2xDB @7 RPE
*Front foot flat. Control the descent. This is your court balance work.

STRENGTH BLOCK C — POSTERIOR + HIP | 3 Rounds
Perform C1 → C2. Rest 60 sec after C2.

C1 — Hip Thrust 10 x 3 sets @barbell @7–8 RPE
C2 — Hip Adductor Machine 15 x 3 sets @7 RPE

8 Cal Air Bike (sprint)

*Reset between rounds — this is power output work, not cardio. Go out at 90% each round.
