STRENGTH BLOCK A — HORIZONTAL PRESS + PULL | 4 Rounds
Perform A1 then A2. Rest 90 sec after A2.

A1 — Bench Press 6 x 4 sets @8 RPE
*Match or add 2.5kg from C2. 2-count descent. Record load.

A2 — Barbell Bent-Over Row 8 x 4 sets @7–8 RPE
*Hinge at 45. Drive elbows back. Bar to lower chest. No hip rise.

STRENGTH BLOCK B — VERTICAL + SHOULDER | 3 Rounds
Perform B1 then B2. Rest 90 sec after B2.

B1 — Weighted Pull-Up 5 x 3 sets @8 RPE
*Full dead hang to chin above bar. Add load when bodyweight feels easy.

B2 — DB Shoulder Press 10 x 3 sets @7 RPE
*Seated or standing. Full lockout. Control the descent.

STRENGTH BLOCK C — CORE + ARMS | 3 Rounds
Perform C1 → C2 → C3 → C4. Rest 60 sec after C4.

C1 — Hanging Leg Raise 10 x 3 sets
*Controlled. No swing. Knees to chest if needed — progress to straight leg.

C2 — Pallof Press 12 x 3 sets (each side) @cable
*Anti-rotation. Brace hard. No trunk lean.

C3 — EZ Bar or DB Curl 12 x 3 sets @7 RPE
C4 — Tricep Pushdown 12 x 3 sets @7 RPE
