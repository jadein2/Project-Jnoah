OLYMPIC LIFTING BLOCK — POWER CLEAN | 4 Sets
Rest 2–3 min between sets.

Power Clean 3 x 4 sets — build from empty bar
*Cycle 3 baseline — first time Power Clean is the main lift. Start at the empty bar or a load that lets you move with speed. Bar starts below the knee. Hip contact before the pull — not arms. Shrug, elbows punch through fast. Catch at quarter squat. Step 1 is positions, not weight.
*Build set to set only if mechanics stay clean. Log every set load.

STRENGTH BLOCK A — FRONT SQUAT + HIP ABDUCTION | 3 Rounds
Perform A1 then A2. Rest 90 sec after A2.

A1 — Front Squat 5 x 3 sets @7–8 RPE
*Elbows stay up — this is the whole session. If elbows drop, bar drops. High bar, clean rack position. Drive knees out. Full depth. Heels down. Cycle 3 baseline — log the load.

A2 — Hip Abductor Machine 15 x 3 sets @7 RPE
*Seated upright. Full range. Slow controlled return — do not let it slam back.

*Short and technical. The bar stays loaded from the Olympic block — strip to a comfortable conditioning weight. HPC mechanics must stay clean under fatigue — hip contact, elbows through. Front Rack Lunges: elbows up even when the legs are burning.
