STRENGTH BLOCK A — PUSH PRESS + ROW | 3 Rounds
Perform A1 then A2. Rest 90 sec after A2.

A1 — Push Press 5 x 3 sets @7–8 RPE
*Cycle 3 baseline. Dip shallow — not a deep squat. Drive with the legs first, the arms finish. Full lockout overhead — active shoulders, push the ceiling. Control the descent back to the rack. Log the load.

A2 — Pendlay Row 8 x 3 sets @7–8 RPE
*Bar starts on the floor between every rep — this is not a touch-and-go. Reset the brace, pull explosively to the lower chest. Upper back drives the movement. Flat back throughout.

GYMNASTICS BLOCK — TOES-TO-BAR SKILL | 3 Rounds
B1 — Toes-to-Bar / Knees-to-Chest 5 x 3 sets
*New movement — introduce the hang first. Grab the bar in a dead hang, create a hollow body position (ribs down, slight posterior pelvic tilt). Pull the knees to the chest with control — no kipping yet. If the knees-to-chest feels solid, try toes to bar. Keep the movement strict. This is skill acquisition, not volume.

METCON

*Descending chipper — starts with the aerobic piece, finishes with the skill piece. Pace the row at 85% — not a sprint, you have a lot of work ahead. Air Squats should be unbroken. DB Push Press and HPC use the same DBs — keep them close. Break the TTB/KTC into sets of 3–5 if needed. The last 10 reps should feel earned.
