# 6FIX — Trainerize Auto-Message Cadence
**Upload to:** Trainerize → Master Program → Auto-messages
**Total messages:** 42 (one per day — days with no message are rest days for the inbox)
**Tone:** Coach Jap voice — direct, calm, no hype, no motivational fluff
**Last updated:** 2026-05-26

---

## How to load in Trainerize

1. Open the 6FIX Master Program in Trainerize
2. Go to Auto-messages → Add message
3. Set trigger: Day number (exact day listed below)
4. Paste message text
5. Set delivery: In-app notification + message

Messages not listed for a specific day = no message that day. Do not fill every day — inbox silence on rest days is intentional.
