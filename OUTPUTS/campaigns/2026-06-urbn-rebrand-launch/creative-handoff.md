# URBN Athletics June 2026 Creative Team Handoff

**Purpose:** Canva-ready creative handoff for June 2026 campaign assets.  
**Brand:** URBN Athletics  
**Core line:** Built to Move Better  
**Main visual rule:** Clean black and white, real gym photography, strong spacing, one clear CTA per asset.  
**Do not use:** Cheap gym promo style, hardcore intimidation, random WOD language, quick-fix fat loss claims, or official HYROX claims.

---

## Canva Setup

Use these rules across all designs:

- Format: Instagram/Facebook square post first, then resize for stories if needed.
- Layout: strong headline, short supporting text, one CTA.
- Brand mark: use `URBN Athletics` text lockup until final logo is approved.
- Tagline: use `Built to Move Better` when it supports the message.
- Photo direction: use real coaches, members, facility, training, or assessment photos.
- Text length: keep each design readable in 3 seconds.

Suggested Canva text hierarchy:

1. Small top label
2. Main hook / headline
3. Short support line
4. CTA
5. Footer: `URBN Athletics | Built to Move Better`

## Creative QA Before Posting

Check every asset before posting:

- Uses `URBN Athletics`
- Uses `Built to Move Better` when relevant
- Has one CTA only
- Does not look like a cheap sale poster
- Does not feel intimidating to beginners
- Uses real gym or coaching context where possible
- Does not imply official HYROX affiliation
- Does not offer Free Trial for URBN FIT
- Keeps price copy clean and not desperate
- Text is readable on mobile
