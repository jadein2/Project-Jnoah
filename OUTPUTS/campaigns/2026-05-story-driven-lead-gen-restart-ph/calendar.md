## How this becomes a SCHEDULED deliverable (the routing step)

Everything above is produced and sitting in `/OUTPUTS/`. To turn it into dated, tracked deliverables, the campaign engine pushes each asset to ClickUp:

1. Confirm/create ClickUp list: `[ACTIVE] 2026-05 Story-Driven Lead Gen — Restart PH` in the CAMPAIGNS folder
2. Each asset → one ClickUp task (name, description = copy + brief reference, due date per this calendar, tags = C/C/V + platform + asset type)
3. Due dates assigned Mon–Sun per the table above
4. Morning briefing then surfaces each day's task automatically
5. When you mark a task Done, it logs to `/GOALS/posting-log.md` as PUBLISHED

**This push is an external action — it writes to your ClickUp workspace. I'll only run it once you say go.**

After one validated week, this same calendar can be handed to `/schedule` to auto-generate the next week's rotation every Sunday.
