Workflow and format rules for all carousel post design and copy.

Source: `social_media_content_design_ruleset.md` + `restart_fitness_ph_content_brain_architecture.md`

Load this for every carousel task — regardless of brand. Always load the relevant brand file alongside this engine.

## Pre-work: route first

Before building any carousel, complete post-routing.md first.

Confirm:
- Brand (Restart Fitness PH / Urban Strong Fitness)
- C/C/V category (Conversion / Connection / Value)
- Content pillar
- One main idea
- One CTA

If any of these are missing, do not start slide copy. Resolve them first.

## Carousel objective

Carousels are the correct format for:
- Education
- Frameworks
- Step-by-step guidance
- Myth vs fact posts
- Client story breakdowns
- Objection handling
- Program explanation
- Saveable content
- Swipe-through commitment

If the idea can be said in one image, use a single image post instead.

## Slide-by-slide structure

### 10-slide structure (educational)

| Slide | Role | Rules |
|---|---|---|
| 1 | **Hook** | 5–10 words max · one strong promise, problem, belief, or curiosity gap · large readable text · one clear visual focus |
| 2 | **Problem** | Name the specific pain or situation · make the audience feel seen · no solution yet |
| 3 | **Why it happens** | Explain the root cause · one reason only · coach-led language |
| 4 | **Common mistake** | What people do instead · frame without shaming |
| 5 | **Better approach** | The shift in thinking or strategy · still one idea |
| 6 | **Practical step 1** | First concrete action · specific, not vague |
| 7 | **Practical step 2** | Second concrete action · builds on step 1 |
| 8 | **Coaching insight** | What a coach sees that most people miss · builds authority |
| 9 | **Summary** | One-line recap of the main idea · not a list of everything |
| 10 | **CTA** | Tell the reader exactly what to do next · one CTA only |

### 8-slide structure (simple idea)

| Slide | Role |
|---|---|
| 1 | Hook |
| 2 | Problem |
| 3 | Why it happens |
| 4 | Common mistake |
| 5 | Better approach |
| 6 | Practical step |
| 7 | Coaching insight |
| 8 | CTA |

## Slide copy rules

### Hook slide (Slide 1)
- 5–10 words maximum
- One strong promise, problem, belief, or curiosity gap
- No explanation — earn the swipe, not explain the post
- Text must be readable without zooming on a phone screen
- One clear visual focus — do not crowd the hook slide

Strong hook patterns:
- Problem statement: "Why you quit every fitness routine after 4 weeks"
- Belief challenge: "You don't need more motivation. You need this."
- Curiosity gap: "The real reason your training stopped working"
- Direct outcome: "How to train 3x a week and actually see results"

Weak hook patterns (avoid):
- "Here are 5 tips for better fitness"
- "Let's talk about consistency"
- Starting with the brand name or program name

### Body slides (Slides 2–9)
- One idea per slide — only one
- 1 to 3 short lines per slide when possible
- Short bullets only — no paragraphs
- Each slide must be understandable without reading the caption
- Do not repeat what the previous slide already said
- Use bold for the key word or key line when design allows

### Final slide (Slide 10 or 8)
- Clear, direct CTA
- Tell the reader exactly what to do next
- Use comment or DM triggers when appropriate
- Brand mark visible on this slide
- Do not add new information — this slide closes, not teaches

## Caption rules

| Spec | Value |
|---|---|
| Length | 800–1,500 characters for educational carousels |
| First 125 characters | Strong hook — this is what shows before "more" |
| Paragraph length | 1–3 lines max |
| CTA position | End of caption |

Caption structure:
1. Strong first line (hook — different from Slide 1 text, not a copy-paste)
2. Brief problem framing (1–2 lines)
3. What the carousel teaches (1–2 lines — tease, not repeat)
4. One CTA

Approved caption CTA examples:
- Comment GUIDE and I'll send you the next step.
- DM "RESTART" if you want a coached system.
- Comment RESET if you want help starting.
- Save this before your next training week.
- Book your Free Assessment — link in bio.

**Caption rule:** The caption adds context the slides don't explain. It does not repeat the slides word for word.

## Design rules

### Core visual rule
The image earns attention. The caption creates understanding. The CTA creates action.

Do not load the slides with explanation. Use the slides to move the reader forward. Use the caption to add depth.

### 2-second test
Every slide must pass this before approving:
- Can the viewer understand the slide's idea quickly?
- Is the headline readable on a phone screen?
- Is there one clear focal point?
- Is there enough negative space?
- Does the eye move naturally through the slide?

If no to any — simplify.

### High contrast rule
- Strong dark/light contrast on every slide
- Clear subject separation
- Bold typography where needed
- One focal point per slide
- No competing visual elements

### Text on slides
- Use the least amount of text needed to carry the idea
- Never use paragraphs on slides
- Never use more than 3 short lines of body copy on a content slide
- Headline must be large enough to read without zooming

### No clutter rule
- No decorative graphics that don't serve the message
- No excessive divider lines
- No random gradients
- No multiple competing messages per slide

## Brand-specific design direction

### Restart Fitness PH carousels
- Background: `#1A1A18` charcoal (primary) or `#F5F5F0` off-white (editorial)
- Text: `#F5F5F0` on dark · `#1A1A18` on light
- Accent: `#E06D22` orange — one emphasis word or CTA only
- Gold `#C9A84C` — proof, results, authority slides only
- Font: Cormorant Garamond Italic for headlines · DM Sans 300/400 for body
- Style: minimalist editorial · no lines · spacing-based separation
- Photography: gritty, high contrast, cinematic, real coaching moments
- No gradients · No decorative clutter · No-Line Rule applies

### Urban Strong Fitness carousels
- Background: `#111111` Urban Black (primary) or `#F1FAEE` Clean White
- Text: `#F1FAEE` on dark · `#111111` on light
- Accent: `#0292b7` Urban Blue — CTA, emphasis, energy moments
- Neutral: `#514d52` Steel Gray — body copy, secondary elements
- Font: Montserrat Bold for headlines · Open Sans Regular for body
- Style: high-contrast, gritty, coach-led, practical
- Photography: real people training · real coaching moments · group energy · beginner-friendly
- No intimidating CrossFit-heavy visuals for cold traffic
- Avoid jargon in cold-traffic slides

## QA checklist — run before any carousel ships

### Slide QA
- [ ] Hook slide: 5–10 words, one idea, one clear visual focus?
- [ ] Every body slide: one idea only?
- [ ] No paragraphs on any slide?
- [ ] Final slide: clear CTA, brand mark visible?
- [ ] Every slide passes the 2-second test?
- [ ] High contrast on every slide?
- [ ] Consistent design across all slides?
- [ ] Brand colors and fonts correct?
- [ ] No clutter, no decorative dividers, no gradients?

### Caption QA
- [ ] First 125 characters create a strong hook?
- [ ] Caption adds context — does not repeat slides word for word?
- [ ] Short paragraphs throughout?
- [ ] One CTA at the end?
- [ ] CTA matches post objective?
- [ ] Tone matches brand voice?

### Copy QA
- [ ] No banned phrases (see GLOBAL/anti-ai-output-rules.md)?
- [ ] Anti-AI Writing Protocol applied?
- [ ] One main idea throughout?
- [ ] Sounds like a coach, not a template?
- [ ] Readable on mobile without zooming?

### Canva workflow (established 2026-05-17)

1. Write full carousel output file → save to `/OUTPUTS/carousel/YYYY-MM-DD-brand-topic.md`
2. Create new design from brand template using template ID above
3. Start editing transaction on the new design
4. Replace all text elements across all slides in one bulk operation
5. Confirm changes with Jap → commit transaction
6. Hand Jap the edit URL — he swaps photos, deletes unused slides, publishes

**What I can edit in Canva:**
- All text (headlines, body, overlines, CTAs) across all slides
- Design title
- Text formatting (color, size, weight, style) — not font family

**What I cannot do in Canva:**
- Place or generate images
- Change font families
- Add or delete slides
- Reposition layout elements
