## System Overview

6FIX is a 6-week habit, nutrition, and accountability coaching program under Restart Fitness PH. It is the entry-level offer in a three-phase coaching ecosystem: 6FIX Phase 1 → Phase 2 → Restart 90.

Core philosophy: Results = Fitness + Nutrition + Accountability.
Core message: You don't need more motivation. You need better decisions, with coaching support.
