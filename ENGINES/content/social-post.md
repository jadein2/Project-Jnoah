Workflow and format rules for all social media captions, single image posts, reels, and stories.

Source: `social_media_content_design_ruleset.md` + `restart_fitness_ph_content_brain_architecture.md`

Load this alongside the relevant brand file and post-routing.md for every social post task.

## Pre-work: route first

Before writing any caption or post, complete post-routing.md.

Confirm:
- Brand
- C/C/V category
- Content pillar
- Platform
- One main idea
- One CTA

Do not write without these resolved.

**Research check:** Check `/RESEARCH/INDEX.md`. If a research file exists for this topic, read Layer 3 (pattern data) for what's resonating on the specific platform right now, and Layer 5 (hypothesis) for pre-seeded content angles. Research is optional for quick posts — recommended for Value and Conversion posts.

## Production workflow — 8 steps

### Step 1: Define the post goal
Choose one: Awareness · Engagement · Education · Trust proof · Lead generation · Offer promotion · Retention · Community building

### Step 2: Define the main message
Write one sentence: What should the audience understand after seeing this post?
If the post has more than one main message, split it into multiple posts.

### Step 3: Choose the best format
- Carousel → education, frameworks, step-by-step, objection handling
- Single image → simple reminder, announcement, quote, belief shift
- Reel → reach, emotional hook, coaching footage, quick insight
- Story → daily presence, DM engagement, polls, warm audience nurturing

### Step 4: Write the image text
- One idea only
- Strong hook
- Minimal words
- Big readable font
- Clear contrast
- No clutter

### Step 5: Write the caption
- First line must hook
- Short paragraphs throughout
- Add context the image doesn't explain
- End with one CTA

### Step 6: Design the visual
Check: one focal point · face or subject clear · headline readable · CTA visible · enough negative space · follows brand style

### Step 7: Add CTA
One CTA only. See CTA selection by intent below.

### Step 8: QA before publishing
Run Design QA and Copy QA checklists at the bottom of this file.

## Platform rules

### Instagram

**Best for:** Carousels · reels · proof posts · identity building · DM conversion

**Caption rules:**
| Type | Length |
|---|---|
| Simple engagement posts | 138–150 characters |
| Educational carousel posts | 800–1,500 characters |
| Maximum allowed | 2,200 characters |

**Image text:** Under 10 words on single images

**Caption structure:**
1. Direct hook (first line — must hook before "more")
2. Short context (1–2 lines)
3. CTA (one only)

**Post rules:**
- Prioritize 4:5 ratio at 1080 × 1350 px
- Hook must be readable immediately
- Use one idea per slide (carousels)
- Short captions for simple posts · longer for education
- End with one CTA

---

### Facebook

**Best for:** Local community trust · offers · longer relationship-building posts · questions · client stories · announcements · lead generation

**Caption rules:**
| Type | Length |
|---|---|
| Simple posts | 40–80 characters |
| Business posts | Under 250 characters when possible |
| Maximum allowed | 63,206 characters |

**Image text:** Short and simple

**Good Facebook hooks:**
- Are you restarting every Monday?
- What is your biggest fitness struggle right now?
- Busy schedule? Start with 3 days.
- Need structure? Start here.

**Post rules:**
- Use question hooks
- Local credibility matters for Urban Strong — use local language and context
- Image text stays minimal
- Longer posts allowed for relationship-building — but only when the topic earns it

---

### Reels and TikTok

**Best for:** Reach · awareness · quick coaching insights · training clips · proof content · relatable pain points · coach authority · transformation storytelling

**Caption rules:**
| Platform | Length |
|---|---|
| Reels | 50–150 characters (longer only for education) |
| TikTok | 50–150 characters |

**On-screen text:**
- Under 20 words per scene
- Strong hook in first 1–3 seconds
- Use captions or subtitles
- Keep text large and readable
- Use keywords that match the topic

**Reel structure:**
1. 1–3 second hook
2. Problem or pattern
3. Simple correction or insight
4. CTA

**Example reel caption:**
"You don't need a harder workout. You need a better system. DM 'RESTART'."

---

### Instagram Stories

**Best for:** Daily presence · polls · quick reminders · behind-the-scenes · community proof · lead nurturing · soft selling · DM starts

**On-screen text:** Under 20 words per frame · one idea per story

**Story structure:**
1. Hook
2. Relate (connect to the audience's situation)
3. One useful practice
4. Poll, question, or interaction sticker
5. CTA

**Good story CTAs:**
- Tap yes if this is you
- Reply RESTART
- Vote below
- Ask me anything
- Want help? Message me
- Book your Free Assessment

---

### LinkedIn

**Best for:** Professional credibility · leadership positioning · health and productivity angles · corporate wellness · business-owner health content · coaching authority

**Caption rules:**
- 1,200–1,600 characters · 200–250 words
- Clean line breaks
- Professional tone
- Expert visual layout
- Image text must feel credible and clean — not loud or gimmicky

**Best for Urban Strong:** Corporate wellness angle, Subic Bay employer outreach, professional credibility content.

---

### X / Threads

**Best for:** One-liners · belief shifts · short coaching lessons · strong opinion · conversation starters

**Caption rules:**
- X: 71–100 characters · keywords first · one idea · strong opinion or clear takeaway
- Threads: Similar to X — one idea, worth saving or replying to, avoid generic motivational lines

## Caption structure templates

### Long caption (educational post)
1. Strong first line — hook before "more"
2. Clear problem (2–3 lines)
3. Coaching insight or explanation (3–5 lines)
4. One practical action (1–2 lines)
5. One CTA

### Short caption (simple post)
Line 1: Direct hook
Line 2: Short context
Line 3: CTA

### Reel caption
One punchy line + one CTA.
Example: "You don't need more workouts. You need a plan. DM 'RESTART'."

## CTA selection by intent

### Engagement CTAs
- Comment below
- Tag a friend
- Save this post
- Share this with someone who needs it

### Lead generation CTAs
- DM "RESTART"
- Comment GUIDE
- Book your Free Assessment
- Apply for the 6-Week Reset
- Comment RESET
- Comment PLAN
- Comment START

### Retention CTAs
- Bring a friend
- Show up today
- Check your plan
- Message your coach

**Rule:** Match CTA temperature to audience temperature. Cold traffic → softer engagement CTA. Warm traffic → direct booking CTA.

## Anti-AI writing protocol

Apply all 10 rules from `/GLOBAL/anti-ai-output-rules.md` before finalizing any caption.

Key behavioral rules for social copy:
- Start from a real human context (observation, question, mistake, pattern)
- Avoid generic AI openers
- Write like a coach explaining something to a real person
- One idea per post — do not stack lessons
- Add specificity: generic → replace with concrete example
- Do not over-polish — it should sound like a person, not a poster
- Keep CTA natural — it should not feel bolted on

## Brand-specific messaging direction

### Restart Fitness PH social posts

Lead with: system · clarity · fundamentals · habits · accountability · real progress · long-term change · structured coaching

Avoid: quick fixes · magic results · extreme transformation claims · shame-based hooks · aggressive "no excuses" messaging

Design direction: minimalist editorial · dark charcoal · off-white text · orange accent · clean spacing · high-contrast photography · calm professional authority

Voice: clear, calm, professional, direct, structured, supportive, practical, human, coach-led

---

### Urban Strong Fitness social posts

Lead with: coached training · structure · fat loss (when campaign-relevant) · strength · consistency · beginner-friendly support · Free Assessment · 6-Week Reset · accountability · local trust

Avoid: intimidating CrossFit-heavy tone for beginners · jargon in cold traffic · random workout-only messaging · discount-first positioning

Design direction: real coaching photos · real members · group energy · beginner-friendly training visuals · clean, strong, practical layouts · clear local CTAs

Voice: direct, grounded, local, confident, approachable, action-oriented, short punchy sentences

## 4-Layer prompt stack

Use this when generating social post content with AI assistance.

### Layer 1 — Brand Resolver
```
Search the project files for the correct brand identity.
If the brand is clear, use it.
If multiple brands are possible, ask which brand to use.
If no brand is provided, default to Restart Fitness PH.
Load the brand's voice, audience, offer, CTA, design rules, banned words, and content priorities before creating output.
```

### Layer 2 — Content Router
```
Route this content request.

Brand:
Audience:
Pain trigger:
Message intent:
Format:
Platform:
Campaign context:

Output:
- Selected brand mode
- C/C/V category: Conversion / Connection / Value
- Content pillar
- Best format
- One main idea
- One practice
- One CTA
- Notes before writing
```

### Layer 3 — Output Generator (Caption)
```
Write a caption for [platform].

Post type: [Carousel / Single image / Reel / Story / Facebook post]
Audience: [Describe audience]
Goal: [Engagement / Education / Lead generation / Trust proof]
Main idea: [Insert idea]
Tone: [Professional, direct, coach-like, beginner-friendly]
Length: [Short / 138–150 characters / 800–1,500 characters]
CTA: [Insert one CTA]

Rules:
- Start with a strong first line
- Use short paragraphs
- Keep the message clear
- Do not over-explain
- Do not use hype language
- End with the CTA
- Apply anti-AI writing protocol
```

### Layer 4 — QA Pass
```
Check the output for:
- Correct brand identity
- One audience
- One idea
- One CTA
- Clear practice
- Platform fit
- Mobile readability
- Natural human voice
- No banned words
- No generic AI phrasing
- Design/copy alignment
Rewrite if needed.
```

## Design QA checklist

- [ ] Text readable on mobile
- [ ] Hook clear within 2 seconds
- [ ] Image text not overloaded
- [ ] One main message
- [ ] One main CTA
- [ ] Strong contrast
- [ ] Clear focal point
- [ ] Brand colors followed
- [ ] Font hierarchy clean
- [ ] Image supports the message
- [ ] Caption does not repeat image text word for word
- [ ] CTA matches post objective

## Copy QA checklist

- [ ] First line creates attention
- [ ] Caption supports the image
- [ ] No unnecessary long intro
- [ ] No vague CTA
- [ ] No over-explaining
- [ ] No multiple offers in one post
- [ ] Paragraphs are short
- [ ] Tone matches the brand
- [ ] CTA is specific
- [ ] Built for one audience and one outcome
- [ ] No banned words or phrases (GLOBAL/anti-ai-output-rules.md)
- [ ] Anti-AI writing protocol applied

## Testing rules

Only test one variable at a time. Do not change the whole system every week.

Testable variables:
- Hook
- Caption length
- Image with face vs no face
- CTA wording
- Carousel length
- Offer angle
- Visual contrast
- Headline size
- Comment CTA vs DM CTA

Track: saves · shares · comments · DMs · profile visits · booked calls · free assessments · new clients
