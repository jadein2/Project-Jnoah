### Color Logic

- Charcoal is the foundation
- Off-white is the editorial background
- Orange is for action and precision
- Gold is for proof, authority, premium signals, and results

### Color Rules (strict)

- Orange — only where action or emphasis is required
- Gold — only for proof, results, authority, or premium signals
- **Never use orange and gold together on the same element**
- Never use pure white `#FFFFFF` (unless a platform technically requires it)
- Never use pure black `#000000`
- Never use orange as a large background
- Never use more than two dominant colors in one visual
- **No gradients ever. Flat color only.**

### Approved Combinations

| Combination | Use |
|---|---|
| Orange on charcoal | Hero sections, social posts, CTA emphasis |
| Gold on charcoal | Results, proof, authority callouts |
| Charcoal on off-white | Website body, cards, editorial posts |
| Soft gray on charcoal | Body copy on dark backgrounds |

## Typography

### Locked Font Stack

| Role | Font | Style / Weight |
|---|---|---|
| Display / Headlines | **Cormorant Garamond** | **Italic, Light 300** |
| Body / UI / Labels | **DM Sans** | 300, 400, 500 |

### Typography Rules

- Cormorant Garamond Italic is the identity font. Use for headlines, campaign words, pull quotes, section titles.
- **Do not use upright Cormorant for Restart Fitness PH headlines. Italic only.**
- Do not use DM Sans for major display headlines.
- DM Sans 300 → body paragraphs. DM Sans 400 → general text. DM Sans 500 → buttons, UI labels, overlines, tags, navigation.
- No more than two font weights in a single layout.
- Generous, editorial line height.
- Retired fonts — do not use: Playfair Display · Montserrat · Bodoni · Inter

### Type Scale

| Role | Size | Use |
|---|---:|---|

## Layout and Spacing

**Base grid:** 8px system.

### Spacing Scale

| Value | Use |
|---:|---|
| 8px | Inline gap |
| 16px | Small element gap |
| 24px | Headline to body gap |
| 32px | Body to CTA gap |
| 48px | Component spacing |
| 64px | Social post margin |
| 96px | Website section padding |
| 128px | Large hero section padding |

### Layout Rules

- Use spacing, not lines, to separate sections
- Use typography hierarchy before adding decoration
- One clear hierarchy per asset
- One CTA per asset
- Preferred layout proportion: **1 : 1.618** (golden ratio). Common split → 38.2% / 61.8%

---

## The No-Line Rule

Use spacing, typography scale, contrast, and layout to separate sections — not divider lines, borders, boxes, or visual clutter.

A small 2px orange accent line is allowed only as a controlled brand accent. Never as a decorative divider or section separator.
